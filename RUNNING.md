# تشغيل موقع ENC (النسخة الكاملة)

هذا الملف يوضح تشغيل الموقع الكامل المبني بـ PHP وربطه بقاعدة البيانات، **وليس** صفحة الـ landing/coming soon.

## المتطلبات

- PHP 7.4+ (أو أي إصدار يدعم `mysqli`)
- خادم ويب (Apache/Nginx) أو استخدام السيرفر المدمج في PHP
- قاعدة بيانات MySQL/MariaDB

## مكان إعدادات قاعدة البيانات

ملف الاتصال موجود هنا:

- `@inc/conn.php`

ويحتوي المتغيرات التالية:

- `$servername`
- `$username`
- `$password`
- `$dbname`

عدّل القيم لتطابق بيانات قاعدة بياناتك.

> ملاحظة: المشروع مضبوط على ترميز `cp1256` ثم يعيد `SET NAMES utf8` لاحقًا داخل نفس الملف. حافظ على هذه الإعدادات إن كانت قاعدة البيانات الحالية تعتمد ترميز عربي قديم، أو عدّلها بما يتوافق مع قاعدة بياناتك.

## تشغيل محلي سريع (بدون Apache)

من داخل مجلد المشروع:

```bash
php -S localhost:8000
```

ثم افتح المتصفح على:

```
http://localhost:8000/index.php
```

## تشغيل عبر Apache/Nginx

1. اجعل جذر الويب (DocumentRoot) يشير إلى مجلد المشروع بالكامل.
2. تأكد من تمكين PHP وربط الامتداد `mysqli`.
3. افتح `index.php` مباشرة بدل `index.html`/`demo-*`.

## ربط قاعدة البيانات

1. أنشئ قاعدة بيانات جديدة (مثال):
   ```sql
   CREATE DATABASE enc_site CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
   ```
2. أنشئ مستخدمًا بصلاحيات كاملة على هذه القاعدة.
3. حدّث قيم الاتصال داخل `@inc/conn.php`.
4. **استيراد الجداول والبيانات:**
   - لا يوجد ملف dump أو schema داخل الريبو الحالي.
   - يلزمك الحصول على نسخة قاعدة البيانات من مصدر المشروع الأصلي ثم استيرادها عبر:
     ```bash
     mysql -u <user> -p <db_name> < dump.sql
     ```

## صفحات الموقع الأساسية (غير صفحة الترحيب)

ابدأ من `index.php`، وهو يقوم بضمّ المكونات الأساسية للموقع عبر ملفات `@inc/*.php`. من هناك يمكنك الوصول إلى الصفحات الأخرى مثل:

- `about.php`
- `service.php`
- `solutions.php`
- `contactus.php`
- `partner.php`
- `dashboard.php` (لوحة التحكم)

> تنبيه: الملفات `index.html` و `demo-*.html` تخص نسخة الـ coming soon وليست الموقع الأساسي.
